# 정산서 완성 검증

## 적용한 안내 문구 (정산서_양식.xlsx A2)
> ※ 집계 규칙: 취소 거래(음수 금액)는 제외하고, 각 기입 금액은 십원 미만 절사하여 기입합니다.

## 검증 결과
- 원장 입력: `ledger.xlsx`를 Python `openpyxl`로 읽음
- 양식 보존: 시트 이름 `월별정산서` 유지
- 노란 입력 칸: 36개 중 36개 채움
- 결과 파일: `정산서_완성.xlsx`
- 재오픈 검증: 아래 최종 검증 스크립트로 정상 재오픈 및 값 확인 완료
- 취소 거래: 음수 금액 거래 제외
- 절사: 각 월별 거래처 집계 금액을 10원 단위로 내림
