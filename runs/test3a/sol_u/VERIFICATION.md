# VERIFICATION — DANTE LABS Earth vs Moon Gravity Lab

## 실행 방법

이 데모는 빌드 단계가 없는 단일 파일 Three.js 페이지입니다.

1. 이 폴더에서 로컬 HTTP 서버를 실행합니다.
   ```bash
   python -m http.server 8000
   ```
2. 브라우저에서 `http://localhost:8000/index.html`을 엽니다.
3. 첫 실행에는 CDN의 Three.js/GLTFLoader/텍스처/GLB 모델을 내려받기 위한 인터넷 연결이 필요합니다.

`file://`로 직접 열 수도 있지만, 브라우저의 CORS 정책 차이를 피하려면 HTTP 서버 실행을 권장합니다.

## 조작법

- `동시에 점프` 버튼 또는 `Space`: 지구와 달의 우주비행사가 같은 초기 속도로 동시에 점프
- `↺` 버튼 또는 `R`: 즉시 초기화
- `Ⅱ/▶` 버튼 또는 `P`: 일시 정지/재개

점프 진행 중에는 중복 점프가 차단됩니다. 두 우주비행사가 모두 착지하면 다시 점프할 수 있습니다.

## 물리 모델 및 기대값

공기 저항을 제외한 수직 등가속도 운동을 사용합니다.

- 공통 초기 속도: `v₀ = 5.0 m/s`
- 최고점: `h = v₀² / (2g)`
- 최고점에서 지면까지의 낙하 시간: `t_fall = √(2h/g)`
- 같은 높이에서 출발하고 착지하므로 상승 시간과 낙하 시간은 같습니다.

| 환경 | 중력 g | 최고점 h | 상승 시간 | 낙하 시간 | 총 체공 시간 |
|---|---:|---:|---:|---:|---:|
| EARTH | 9.81 m/s² | 1.274209990 m | 0.509683996 s | 0.509683996 s | 1.019367992 s |
| MOON | 1.62 m/s² | 7.716049383 m | 3.086419753 s | 3.086419753 s | 6.172839506 s |

콘솔 출력은 소수점 셋째 자리로 고정됩니다.

```text
[EARTH] jump_apex_m=1.274
[EARTH] fall_time_s=0.510
[EARTH] landed
[MOON] jump_apex_m=7.716
[MOON] fall_time_s=3.086
[MOON] landed
```

참고: 실제 시간순으로는 지구의 세 로그가 먼저 완료되고, 달의 최고점/착지 로그가 뒤에 이어집니다. 브라우저 프레임률에 따른 적분 오차가 출력값에 스며들지 않도록 최고점과 낙하 시간 로그는 닫힌형 해를 사용합니다. 화면 운동은 `1/120초` 고정 물리 스텝으로 계산합니다.

## 성능/안정성 설계

- 하나의 WebGLRenderer에서 scissor viewport 두 개를 렌더링해 캔버스/컨텍스트 중복을 피했습니다.
- 물리 계산은 120 Hz 고정 스텝, 렌더링은 `requestAnimationFrame`을 사용합니다.
- devicePixelRatio를 최대 2로 제한해 고해상도 화면의 GPU 부하를 제어합니다.
- 프레임 간 시간은 최대 50 ms로 제한해 탭 복귀 시 물리 폭주를 방지합니다.
- 먼지 입자는 수명이 끝나면 geometry/material을 dispose합니다.
- 우주비행사 원격 GLB 로딩 실패 시 동작 가능한 로컬 절차적 대체 모델을 사용합니다.
